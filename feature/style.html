jQuery Gridy is a plugin that generates a highly customizable grid using templates.

HTML:
<div id="grid"></div>
Template:
<script id="template" type="text/x-jquery-tmpl">
   <div>
      <div>${name}</div>
      <div>${email}</div>

      <div class="gridy-button">
         <a href="#">edit</a>
      </div>
    /div>
</script>
Script:
style: 'div'
TIP:
- When you use div style, you can build a lot of types of design inside each line, not just table style.
